    QUESTIONS FRÉQUENTES
═══════════════════════════════════════════════════════════════════════

Q : Pourquoi le solde ne change pas après une dépense ?
R : Vérifiez que la dépense est au statut "Validée"
   Seules les dépenses validées sont comptabilisées

Q : Comment voir le détail des dépenses ?
R : Allez dans Finances → Transactions
   Filtrez par type "Dépense"

Q : Le solde est négatif, que faire ?
R : 1. Vérifiez les dépenses
   2. Ajoutez un dépôt client
   3. Contactez le client pour un paiement

Q : Puis-je voir le solde par projet ?
R : Oui ! Allez dans Projets → [Votre Projet]
   Consultez l'onglet "Finances"
